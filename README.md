# Temporal Playground

0. Install [temporal cli](https://temporal.io/setup/install-temporal-cli)
1. Run temporal (blocks console)
```bash
./startTemporal.sh
```
2. Configure temporal
```bash
./configureTemporal.sh
```
3. Install deps
```bash
npm i
```
4. Run worker (blocks console)
```bash
npm run worker
```
5. Run api (blocks console)
```bash
npm start
```
6. Run demo 
```bash
./demo.sh
```
